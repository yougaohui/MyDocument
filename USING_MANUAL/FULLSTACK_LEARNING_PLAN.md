## 🛠️ 技术栈选择

### 前端技术栈
- **框架：** React 18 + TypeScript
- **UI 库：** Ant Design / Material-UI
- **状态管理：** Redux Toolkit / Zustand
- **构建工具：** Vite / Webpack

### 后端技术栈
- **运行时：** Node.js + Express
- **数据库：** PostgreSQL / MongoDB
- **认证：** JWT + OAuth 2.0
- **API 设计：** RESTful + GraphQL

### 移动端技术栈
- **基础：** 你的 Android 开发经验
- **架构：** MVVM / Clean Architecture
- **网络：** Retrofit / OkHttp
- **数据库：** Room / GreenDAO

### DevOps 工具链
- **版本控制：** Git + GitHub
- **CI/CD：** GitHub Actions
- **容器化：** Docker
- **部署：** Vercel / AWS / 阿里云
