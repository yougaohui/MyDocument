## 🚀 核心特色

### 🪄 魔法指令系统 (创新亮点)

| `search` / `find` | 智能搜索模式 | 快速信息查找 | 🔍🔍 |

## 🎮 使用方式

### 新手用户 (5分钟上手)
1. 阅读 `QUICK_START.md`
2. 一键安装: `bunx oh-my-opencode install`
3. 测试基础功能: `ulw: 构建一个简单应用`

### 中级用户 (深入使用)
1. 学习 `README.md` 中的魔法指令
2. 配置高级功能: 参考 `ADVANCED_FEATURES.md`
3. 实践多代理协作: `orchestrate: 设计并实现系统`

### 高级用户 (完全掌握)
1. 自定义配置: 参考 `CONFIG_EXAMPLES/`
2. 优化工作流: 创建自定义工作流模板
3. 集成外部工具: MCP服务器、自定义代理
